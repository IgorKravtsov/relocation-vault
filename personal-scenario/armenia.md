# Армения — персональный разбор

> Дата: **2026-06-20**. Курсы: **$1 ≈ AMD 385**, €1 = $1.157. Доход $5,600/мес ≈ AMD 25.9m/год; $7,500 ≈ AMD 34.6m/год. Валюта — **армянский драм (НЕ евро)**.
> Статус: разбор под профиль (см. [`README.md`](./README.md)). Правила — **проверить у армянского бухгалтера/юриста**.

## 0. Краткий вывод (TL;DR)

- **Налог для IT — потенциально ~1% (turnover tax), уровень Грузии.** IT-деятельность с оборотом до **AMD 120m (~$300k)** облагается **1% с оборота** вместо обычных 5%. При твоём доходе попадаешь с огромным запасом.
- ⚠️ **Нюанс ИП:** для индивидуального предпринимателя (sole proprietor) turnover tax заменяет **только НДС**, а не подоходный → взаимодействие с подоходным надо подтвердить у бухгалтера. Поэтому эффективную ставку закладываю **диапазоном ~1–10%**, пока не подтверждено.
- **Micro-business 0%** (оборот до AMD 24m ≈ $62k) — ты **чуть выше порога** ($67k), так что 0% не проходит; целевой режим — **IT 1%**.
- **Очень просто для украинца:** безвиз **180 дней/год**, ВНЖ по бизнесу/работе, русский язык широко используется.
- Сильная **русскоязычная среда + дешёвая жизнь + IT-комьюнити** (большой приток с 2022). Минусы: **не ЕС**, драм волатилен (резко укрепился на российских деньгах), геополитика региона.

## 1. Налоги

**Резидентство:** 183+ дня → облагаешься как армянский налоговый резидент.

**Режимы (по убыванию выгодности для тебя):**

- **Micro-business — 0%:** оборот ≤ **AMD 24m/год** (~$62k) → освобождение от основных налогов. 🔴 Твои ~$67k чуть выше → **не подходит** (подошло бы при доходе < ~$5,200/мес).
- **IT turnover 1% — целевой:** IT-деятельность, оборот ≤ **AMD 120m** (~$300k) → **1% с оборота** (вместо стандартных 5%). Ты проходишь с запасом.
- **Обычный turnover 5%:** если 1%-льгота по IT не применится к твоему профилю.
- **Общий режим (20% подоходный):** консервативный сценарий, если ни один льготный режим не сработает (так считало первичное досье `countries/armenia.md`).

**Оценка эффективной ставки (диапазон, $5,600/мес):**

| Сценарий | Налог | На руки |
|---|---:|---:|
| IT 1% turnover (оптимистично) | ~1–5% + фикс-взносы | ~**$5,300** |
| Если только 5% turnover | ~5–8% | ~$5,150 |
| Общий 20% (пессимистично) | ~20%+ | ~$4,500 |

🔴 **Главный открытый вопрос — какой режим реально применим к IT-ИП с иностранными клиентами** (turnover tax для sole proprietor заменяет только НДС; нужно понять взаимодействие с подоходным и фикс-взносами). Это решает, Армения «уровня Грузии» или «уровня Румынии». **Обязательно к бухгалтеру до решения.**

**НДС:** услуги иностранным B2B обычно вне армянского НДС (место поставки = страна клиента) — подтвердить.

## 2. Легализация (под украинца)

- **Безвиз 180 дней/год** для украинцев — для короткого старта.
- **ВНЖ (temporary residence)** по основанию: **бизнес-деятельность** (свой ИП/компания), работа, учёба, семья. Для IT-ИП — стандартный путь «регистрация ИП → ВНЖ по предпринимательству».
- **Польская TP не нужна и не мешает** — Армения даёт свой статус с нуля; здесь нет зависимости от 03.2027 (в отличие от ЕС-стран).
- **ПМЖ/гражданство:** есть лестница (temporary → permanent → citizenship); точные сроки и языковой ценз — к юристу.
- 🔴 **Не ЕС** → евро-ПМЖ не строит (как Грузия/Албания). Это чисто накопительная база.

## 3. Девушка / будущая жена

- **Воссоединение семьи** через твой ВНЖ — стандартно для **супруги** (брак).
- Гражданский партнёр без брака как основание — **подтвердить отдельно** (в отличие от Сербии, где гражданский брак прямо признан). Пока считать: **надёжнее брак** либо её собственный ВНЖ (свой ИП/учёба).
- Короткий мост — её собственный безвиз 180 дней.

## 4. Стоимость жизни (Ереван, на двоих)

Numbeo/Livingcost (2026): базовые расходы одного без аренды ~$800–900; аренда 2-комн в центре выросла до ~$1,000 (российский приток 2022–2023 поднял цены).

| Статья | Реалистично (пара) |
|---|---:|
| Аренда 1–2BR | $600–1,000 |
| Коммуналка/интернет/связь | $120–200 |
| Еда/быт | $500–700 |
| Транспорт/такси | $80–150 |

При IT 1% (net ~$5,300) накопление **~$3,000–3,500/мес** → **$90k за ~26–30 мес**. Близко к Грузии.

## 5. Уровень жизни

Плюсы: **русский язык повсеместно**, русско-/украиноговорящие врачи (плюс против Албании), большое IT-комьюнити и инфраструктура для релокантов, дешёвая еда/быт, безопасно, дружелюбно к украинцам.

Минусы: **не ЕС**; **драм волатилен** (резкое укрепление 2022–2023, риск в обе стороны); **геополитика** (соседство с Азербайджаном/Турцией, регион нестабилен); landlocked, нет моря; аренда в Ереване подорожала из-за притока.

## 7. Вердикт (первичный)

Армения — **сильный накопительный кандидат уровня Грузии**, ЕСЛИ подтвердится IT 1%:
- ~1% налог + дешёвая русскоязычная база + очень простой доступ украинцу.
- Минусы: не ЕС, драм волатилен, геополитика региона, нет моря, аренда в Ереване подросла.

**Грузия vs Армения:** обе ~1%, русскоязычные, вне ЕС. Грузия — отлаженный 1% ИП и виза на год; Армения — формально такой же 1% по IT, но **налоговый маршрут для ИП надо распутать** (turnover для sole proprietor заменяет только НДС). Пока Грузия = надёжнее, Армения = эквивалент при подтверждении режима. Для ЕС-ПМЖ — в любом случае отдельная фаза (Греция).

## 8. Открытые вопросы (бухгалтеру/юристу)
1. **Главный:** применим ли IT 1% turnover к ИП с иностранными клиентами; как взаимодействует с подоходным (для sole proprietor turnover заменяет только НДС) и фикс-взносами → реальная эффективная ставка.
2. Подойдёт ли micro-business 0%, если держать оборот < AMD 24m (стоит ли занижать ниже $5,200/мес — вряд ли).
3. НДС/place-of-supply по иностранным B2B.
4. ВНЖ по предпринимательству: точный чек-лист, считается ли в ПМЖ, механика партнёра (брак vs свой статус).
