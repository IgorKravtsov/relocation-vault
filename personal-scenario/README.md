# Personal Scenario

Персональные разборы стран под конкретную задачу: **накопить капитал и/или перебраться в ЕС**, с учётом реальной финансовой и семейной ситуации. В отличие от `countries/` (общие досье на английском по vault-протоколу), здесь — прикладные расчёты под профиль, на русском.

## Профиль

- Гражданин Украины, в Польше по **PESEL.UKR** (временная защита), истекает **март 2027**. Сильной привязки к Польше нет.
- Работает **B2B**, сейчас на **общих основаниях (skala, 12/32%)** — худший польский режим.
- Доход **~$5,600/мес** (с этого месяца; $3,000 + $2,600), реалистичный рост до **~$7,500/мес**.
- Девушка не работает, переезжает вместе. **Готовы заключить брак.**
- Цель: **на этапе накопления — снимать двухкомнатную**; копить капитал максимально быстро; **позже купить 3–4-комнатную** (место — TBD, возможно вне Греции). Параллельно/потом — закрепиться в ЕС (ПМЖ).

## Ключевой вывод (по 7 разобранным странам)

**Две фазы плохо совмещаются в одной стране:** дёшево+низкий налог = вне ЕС; в ЕС = дороже/выше налог. Покупка (3–4 комнаты) — отдельный поздний шаг, поэтому сейчас главная метрика — **скорость накопления капитала**.

Скорость накопления (аренда двушки, реалистичный бюджет, доход $5,600):
| Страна / город | Налог (IT) | На руки | Откладываешь | $90k за |
|---|---|---:|---:|---:|
| **Албания / Дуррес** | **~0% (до 2029)** | ~$5,420 | ~$3,670 | **~25 мес** |
| Черногория / Подгорица | ~0% (DN, сунсет 2026) | $5,600 | ~$3,400 | ~26 мес |
| Грузия / Тбилиси | ~1% | $5,540 | ~$3,440 | ~29 мес |
| Болгария / Пловдив | ~18% | $4,569 | ~$2,420 | ~36 мес |
| Греция / Салоники | ~14% (5C) | $4,794 | ~$2,124 | ~38 мес |
| Кипр | ~17% (компания) | $4,634 | ~$2,000 | ~46+ мес |
| Румыния | ~22–26% | $4,153 | ~$2,050 | ~44 мес |

- **Чистая скорость кэша** → **Албания** (~0% до 2029, тёплое море; проверено deep-research) ≈ **Грузия** (надёжный 1%, без сунсета, проще виза). Албания чуть быстрее + море; Грузия стабильнее. Черногория ≈ та же скорость, но 0% на сунсетящейся (2026) программе.
- **«Накопить + осесть в ЕС в одной стране»** → Греция (№1 в ЕС): ~14% по 5C + уникальный мост ст. 194 (TP→ВНЖ изнутри) + путь партнёрше без брака + тёплый климат/низкая преступность/высокий английский. Минус — дороже и медленнее.
- Болгария снята как ПМЖ-вариант (PR-маршруты тебе недоступны); Кипр дорог/медленен; Румыния — ~22–26% и та же PR-ловушка.
- Все низконалоговые-быстрые (Албания/Грузия/Черногория) — **вне ЕС** → для евро-ПМЖ нужна отдельная фаза в ЕС (Греция).
