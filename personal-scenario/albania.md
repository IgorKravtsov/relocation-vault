# Албания — персональный разбор

> Дата: **2026-06-14**. Курсы: **$1 ≈ ALL 82.5**, **$1 ≈ €0.865**. Доход $5,600/мес ≈ ALL 5.54m/год; $7,500 ≈ ALL 7.42m/год. Валюта — **албанский лек (НЕ евро)**.
> Статус: разбор под профиль (см. [`README.md`](./README.md)). Правила — **проверить у албанского бухгалтера/юриста**.

## 0. Краткий вывод (TL;DR)

- **Налог для IT — ~0% (≈3% эффективно), ПРОВЕРЕНО deep-research (июнь 2026):** Конституционный суд 27.06.2024 отменил 15% на свободные профессии, и его НЕ пере-приняли → IT под общим режимом малого бизнеса **0% PIT при обороте до ALL 14m, до 31.12.2029**. Только фикс-взносы ~$180/мес. (Состязательная проверка убила 0-3 все утверждения про 15%.) Риски: сунсет 2029 + возможное пере-законодательство; подтвердить у бухгалтера.
- Дёшево + тёплое побережье + безопасно, но **НЕ ЕС** (кандидат ~2030) → путь к евро-ПМЖ не строит.
- Unique Permit (DN) тянется до 5 лет → **албанский** ПМЖ возможен (но не ЕС).
- **Застряла посередине:** хуже Грузии (1%) по налогу, не ЕС в отличие от Греции/Болгарии.
- Партнёр: нужен **брак**. Безвиз 90/180 (не год). Лек → валютный риск.

## 1. Налоги

**Резидентство:** постоянный дом или 183 дня → облагается по миру.

**Подтверждено deep-research (июнь 2026; 19 источников вкл. tatime.gov.al, PwC, EY, Eurofast, решение КС; 15/25 фактов подтверждены состязательно, утверждения про 15% убиты 0-3):**
- **0% подоходного** для IT-фрилансера: КС **27.06.2024 отменил** 15% на свободные профессии, НЕ пере-приняли → IT под общим режимом малого бизнеса **0% при обороте до ALL 14m (~€120k), до 31.12.2029**.
- **Соцвзносы фиксированные:** социальные 23% на базу ALL 50,000 + здоровье 3.4% на ALL 100,000 = **~ALL 14,900/мес (~$180)**. Это вся нагрузка.
- **НДС:** услуги иностранным B2B — **вне сферы албанского НДС** (принцип назначения, место поставки = страна клиента); порог ALL 10m — ты под ним. НДС не платишь. (Фискализация счетов всё равно обязательна.)
- **Анти-уклонение** (переквалификация одноклиентского фрилансера в 13%/23%) **НЕ применяется** при обслуживании только нерезидентов без PE → ты в безопасности (держать структуру «только иностранные клиенты»).

**Эффективная ставка ~3% ($5,600/мес ≈ ALL 5.54m/год):**
| | |
|---|---:|
| Подоходный | **0%** |
| Соцвзносы (фикс) | ~$180/мес |
| **Нагрузка (~3%)** | **~$180/мес** |
| **На руки** | **~$5,420** |

🔴 **Риски:** 0% заканчивается **31.12.2029**; 15% один раз уже вводили (отменён судом) → риск повтора; «только иностранные клиенты» критично; подтвердить у бухгалтера.

## 3-bis. КАК оформиться ради 0% — Person Fizik, НЕ «цифровой кочевник»

⚠️ В Албании **два маршрута с противоположным налоговым результатом** — не путать:
- **❌ Unique Permit / DN:** по Закону 25/2022 такой работник **НЕ налоговый резидент Албании 12 мес** → ты вне албанской системы, налоговый резидент «где-то ещё» (Украина/серая зона). НЕ даёт албанский 0% и мутно для AML.
- **✅ Person Fizik (самозанятый):** регистрируешься албанским самозанятым → албанский налоговый резидент → **0% на прибыль** (оборот < ALL 14m, до 2029). Чистый задокументированный 0% — **это и нужно**.

**Шаги (Person Fizik):**
1. Регистрация Person Fizik в QKB через **e-Albania** («initial registration of natural person») — онлайн, ~2 дня, бесплатно → **NIPT**.
2. **ВНЖ на основании самозанятости** (связку «регистрация + permit» сшивает юрист).
3. Налог **0%** (оборот < ALL 14m, до 31.12.2029); годовая декларация (налог = 0).
4. Взносы фикс ~**ALL 12,500–14,900/мес (~$150–180)**.
5. **Фискализация** каждого счёта (отдельная обязанность, 0% не освобождает).
6. НДС: < ALL 10m + иностранные B2B вне сферы → не платишь.

**Чтобы сохранять 0%:** оборот ≤ ALL 14m; только иностранные клиенты (экспорт + защита от переквалификации); фискализация + декларация + взносы; продление ВНЖ; быть налоговым резидентом (183+ дня).

## 4. Легализация

- **Безвиз украинцам — 90/180 дней** (не год). Для долгого — Type D + **Unique Permit**.
- **Unique Permit (DN, пилот с 01.2026):** доход на **основного заявителя** (источники расходятся: от ~€817 до €3,000/мес — единой цифры нет; при $5,600 проходишь с запасом). Иностранные клиенты ОК, нет доступа к местному рынку труда. **1 год, продлевается до 5 лет → возможен ПМЖ** (лучше черногорского 4-летнего капа).
- **Девушка:** включается только как **супруга (нужен брак)**; порог дохода тогда растёт (официально надбавку не публикуют, обычно ~+20–30% — ты всё равно проходишь). Незарегистрированный партнёр не покрыт.
- 🔴 **ПМЖ Албании — НЕ ЕС** (кандидат ~2030, спекулятивно) → путь к ЕС не строит, как Грузия/Черногория.
- TP-моста после 03.2027 нет.
- **Партнёр:** супруг/иждивенец → нужен **брак** (незарегистрированный не подтверждён).

## 5. Качество жизни (детализировано)

**Город (рекомендация):** 🥇 **Дуррес** — баланс: море + дёшево + **40 мин до Тираны** (больницы/аэропорт). **Тирана** — если приоритет сервисы/частная медицина/IT-комьюнити (но дороже, жаркое лето инленд). **Влёра** — теплее/дешевле всех, но мало сервисов и далеко от топ-больниц.

**Аренда двушки (~60–70 м²):** Тирана ~$550–750 · Дуррес ~$400–550 · Влёра ~$400–500. Полный бюджет на пару (с медстраховкой): Дуррес ~$1,800, Тирана ~$2,150.

**Медицина — 🟡 частная + страховка:** публичная слабая (избегать); частная доступная и приличная в **Тиране** (American Hospital — JCI, German Hospital, англоговорящие врачи). Терапевт €30–60, специалист €40–100; частная страховка **€30–150/мес/чел** (на пару ~€60–200). ⚠️ Топ-больницы — в Тиране → аргумент за Тирану/Дуррес, не Влёру.

**Безопасность — 🟢:** US Level 2 (как Франция/Италия/UK); насилие против иностранцев крайне редко; имущественные преступления ниже многих городов UK. Основное — карманники. Безопасные районы Тираны: Blloku, Komuna e Parisit, у Большого парка, центр. Женщине безопасно одной. 🔴 **Главный реальный риск — дороги** (один из худших в Европе уровней смертности; агрессивное вождение).

**Английский / украинцы:** английский умеренный (города/молодёжь ОК, ниже греческого); русскоязычная община маленькая; страна прозападная, к украинцам дружелюбно. Албанский — для ПМЖ/гражданства.

**Климат:** тёплый, солнечный; Дуррес/Влёра — мягкая зима, жаркое лето; Тирана — жарче инленд.

## 5-ter. Бухгалтер — обязателен

Даже при **0%** есть обязанности: **фискализация каждого счёта** (NIVF), **ежемесячные взносы** (~ALL 12,500–14,900), **годовая декларация** (даже с нулём), корректная регистрация Person Fizik + проверка применимости 0% к IT. → **Местный англоговорящий бухгалтер с самого старта**, ориентир **~€50–150/мес**. Бонус: ведёт чистую задокументированную историю дохода (нужно для AML при будущей покупке в ЕС — см. [`buying-property-in-eu.md`](./buying-property-in-eu.md)).

## 7. Вердикт (обновлён после deep-research)

Албания — **топ для фазы накопления**, наравне с Грузией или чуть быстрее:
- Налог для IT — **~0% (≈3% эффективно), проверено**; накопление ~25 мес — быстрее Грузии/Черногории.
- Плюсы: **0% + дешёвое тёплое адриатическое побережье** + Unique Permit до 5 лет → албанский ПМЖ.
- 🔴 Минусы: **сунсет 0% 31.12.2029** + риск повтора 15% (история отмены судом) → менее «вечно стабильно», чем Грузия; **не ЕС** (евро-ПМЖ не строит); лек (валютный риск); безвиз 90/180; брак для партнёрши; английский/медицина слабее греческих.

**Грузия vs Албания:** обе ~0–1%, обе не ЕС. Албания — **тёплое море + 0%**, но 2029-сунсет и риск повтора налога. Грузия — **стабильнее/без сунсета**, проще виза (1 год + border-run), мощнее IT-инфраструктура. Для ~2–3-летнего окна накопления (до 2029) — обе отличны; выбор по приоритету «тёплое море vs надёжность». Для ЕС в любом случае — Греция.

## 8. Открытые вопросы (юристу/бухгалтеру)

2. Реальная база соцвзносов для high-earner (минимум ALL 14,900 — финально?).

4. Unique Permit: точный чек-лист, считается ли в 5-летний ПМЖ, механика партнёра.
