# Босния и Герцеговина — персональный разбор (фаза 1, накопление)

> Дата: **2026-06-20**. Курсы: €1 = 1.95583 BAM (фикс), €1 = $1.157. Доход $5,600/мес ≈ €4,840 ≈ 9,460 BAM/мес; $7,500 ≈ €6,480.
> Статус: разбор под профиль (см. [`README.md`](./README.md)). Правила — **проверить у местного бухгалтера/юриста**. ⚠️ БиГ — две налоговые юрисдикции: **Федерация БиГ** и **Республика Сербская (РС)** + округ Брчко; правила различаются.

## 0. Краткий вывод (TL;DR)

- **Плоский подоходный 10%** во всех энтитетах — но с **обязательными соцвзносами** (по доходным брекетам), так что эффективно для тебя **выше 10%**.
- 🔴 **Интрига 2% Республики Сербской ЗАКРЫТА (2026-06-20): не работает на твоём доходе** — порог BAM 50k/год, выше → НДС + соцвзносы. Базовый сценарий 10% + взносы.
- **Не ЕС** (потенциальный кандидат). Накопительная база вне ЕС.
- **Легализация украинцу — ОК:** ВНЖ через бизнес (компания от €500 капитала), 12 мес, 2–6 недель; TP доступна.
- 🔴 Маленькая русско-/украиноязычная община; континентальный климат, фактически landlocked (выход к морю — 20 км Неум); сложная админка (две юрисдикции).

## 1. Налоги

- **Подоходный — плоский 10%** во всех энтитетах (занятость, бизнес, проценты, роялти) — одна из простейших систем Европы.
- 🔴 **Соцвзносы обязательны** (здоровье/пенсия/соц), считаются по доходным брекетам → эффективная нагрузка для твоего дохода **выше 10%** (точно — к бухгалтеру; ориентир уровня Болгарии/Сербии).
- 🔴 **Республика Сербская — «малый предприниматель» 2% с оборота: для тебя НЕ работает (проверено 2026-06-20).** Режим реален, но при обороте выше **BAM 50,000/год (~€25,500)** включаются обязательная НДС-регистрация + соцвзносы. Твой оборот ~€60–78k/год — в 2.4–3× выше порога → чистые 2% не достаются (та же пороговая ловушка, что «10% без взносов» в Сев. Македонии и paušal в Сербии). Базовый сценарий для тебя — 10% + соцвзносы.
- **НДС:** порог регистрации **~50,000 BAM/год (~€25,500)** — твой оборот выше → формально регистрация; экспорт услуг иностранным B2B обычно вне сферы/reverse-charge — подтвердить.

## 2. Легализация (украинцу)

- **ВНЖ через бизнес:** регистрация компании (d.o.o.) от **€500 капитала** → временный ВНЖ до 12 мес; срок оформления **2–6 недель**. Нужна реальная деятельность (инвойсы, отчётность).
- **TP** для украинцев доступна.
- 🔴 **Не ЕС** → евро-ПМЖ не строит.

## 3. Девушка / партнёрша

- Своя TP либо свой ВНЖ; долгосрочно — воссоединение (вероятно **брак**; гражданский партнёр не подтверждён). Уточнить.

## 4. Стоимость жизни (Сараево, на двоих)

Очень дёшево: один ~€800–1,200/мес; аренда 1BR в центре €300–500, вне центра €200–350. Пара реалистично **~$1,200–1,600/мес**. Уровень Скопье, дешевле Тбилиси. Питание дёшево (ужин на двоих ~$11).

## 5. Уровень жизни

- 🔴 **Климат:** континентальный, холодная снежная зима (Сараево в горах), фактически без моря.
- 🔴 **Русско-/украиноязычная община — маленькая**; врачи по-русски не подтверждены. Слабее Грузии (важно для неработающей партнёрши).
- 🟡 Безопасно, дёшево, красивая природа; английский средний; админка усложнена двумя юрисдикциями.

## 6. Вердикт (фаза 1)

🔴 **Не обгоняет Грузию.** Козырь «2% Республики Сербской» **проверен и закрыт** (2026-06-20): он только для мелких предпринимателей (порог BAM 50k/год), твой оборот втрое выше → базовый сценарий 10% + соцвзносы (уровень Болгарии/Сербии). Плюс маленькая community, континентальный климат, не ЕС. **Для накопления смысла нет** — снято.
