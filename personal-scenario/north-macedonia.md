# Северная Македония — персональный разбор (фаза 1, накопление)

> Дата: **2026-06-20**. Курсы: €1 ≈ 61.5 MKD, €1 = $1.157. Доход $5,600/мес ≈ €4,840 ≈ 298k MKD/мес; $7,500 ≈ €6,480 ≈ 399k MKD/мес.
> Статус: разбор под профиль (см. [`README.md`](./README.md)). Правила — **проверить у местного бухгалтера/юриста**.

## 0. Краткий вывод (TL;DR)

- 🔴 **«Чистые 10% без соцвзносов» — НЕ про твой доход.** Льготный фриланс-режим (10% и без обязательных пенсия/медицина) действует при обороте **до ~90,000 MKD/мес (~€1,500)**. Твои ~€5,000–6,500/мес — **в разы выше** → нужна полноценная регистрация (ИП/компания) с соцвзносами. Реальная эффективная ставка для тебя — ориентировочно **~10% подоходный + соцвзносы ~28% (с базы)** = заметно выше «красивых 10%».
- **Не ЕС** (кандидат). Накопительная база вне ЕС, как Грузия/Албания, но **без их преимуществ** (налог выше 1%/0%; community меньше).
- **Легализация украинцу — ОК:** TP до 04.03.2027; self-employment ВНЖ (директор/учредитель македонской фирмы) — 1 год, **продлевается без ограничения срока**, без теста рынка труда.
- 🔴 Маленькая русско-/украиноязычная община; континентальный климат (нет моря, холодная зима); landlocked.

## 1. Налоги

- **Подоходный — плоский 10%** на самозанятость (с 01.2023), в т.ч. доход от иностранных клиентов.
- 🟢 **Льгота для мелких фрилансеров:** при обороте **≤ ~90,000 MKD/мес (~€1,500)** — только 10%, **без обязательных соцвзносов**. Это и есть «10% без взносов», о которых пишут.
- 🔴 **Но твой доход ~€5,000–6,500/мес >> €1,500** → льготный потолок не проходишь → регистрируешь ИП/самозанятость/компанию **с обязательными соцвзносами** (пенсия+медицина ~27–28% с базы). Эффективная ставка тогда — **выше 10%**, точную считать с бухгалтером (зависит от базы взносов и формы).
- **НДС:** порог регистрации ~2M MKD/год; экспорт услуг иностранным B2B обычно вне сферы/reverse-charge — подтвердить.

**Вывод по налогам:** на твоём доходе это **не «уровень Грузии»**, а скорее уровень Болгарии/Сербии (~12–18% эффективно с взносами). Главное преимущество страны (10% без взносов) до тебя не дотягивается.

## 2. Легализация (украинцу)

- **TP до 04.03.2027** (гуманитарный ВНЖ даёт право работать без отдельного permit).
- **Self-employment ВНЖ:** для учредителя/директора македонской компании — 1 год, **продление без ограничения по сроку**, без labour market test. Подходит под IT-фрилансера через свою фирму.
- ⚠️ Существующий **польский karta pobytu** даёт только краткий безвиз (15 дней/въезд, ≤90/180) — не замена ВНЖ.
- 🔴 **Не ЕС** → евро-ПМЖ не строит.

## 3. Девушка / партнёрша

- Своя **TP** (до 03.2027) — независимо от тебя.
- Долгосрочно — воссоединение семьи (вероятно нужен **брак**; гражданский партнёр не подтверждён) либо свой ВНЖ. Уточнить.

## 4. Стоимость жизни (Скопье, на двоих)

Дёшево: Numbeo/Livingcost 2026 — семья ~$1,292/мес; аренда 1BR от ~$229. Пара реалистично **~$1,200–1,600/мес**. Дешевле Тбилиси.

## 5. Уровень жизни

- 🔴 **Климат:** континентальный — жаркое лето, **холодная зима, моря нет**; Скопье зимой — смог (один из худших в Европе по загрязнению воздуха).
- 🔴 **Русско-/украиноязычная община — маленькая** (нет данных о значимой диаспоре), врачи по-русски — не подтверждено. Слабее Грузии по этой оси (важно для неработающей партнёрши).
- 🟡 Английский средний; страна дружелюбна, безопасна, очень дёшева.

## 6. Вердикт (фаза 1)

🔴 **Не обгоняет Грузию.** На твоём доходе теряется главный козырь (10% без взносов — только для мелких фрилансеров), остаётся ~12–18% эффективно — хуже Грузии (1%) и Албании (0%). Плюс маленькая community, континентальный климат, не ЕС. Единственный плюс — очень дёшево жить. **Для накопления смысла мало.**

## 7. Открытые вопросы (бухгалтеру/юристу)

3. Партнёрша: брак vs свой ВНЖ; считается ли что-то в ПМЖ.
