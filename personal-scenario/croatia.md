## 0. Краткий вывод (TL;DR)

- **ЕС + еврозона + Шенген** (с 2023), **тёплое Адриатическое море**, **хорватский язык славянский → учить легко**.
- 🟢 **Налог на базовом доходе может быть низким** через paušalni obrt (~9–13%), НО только до оборота **€60,000/год** — твой $5,600 (~€58k) впритык влезает, $7,500 (~€78k) — нет.
- 🔴 **Для паспорта слабее Кипра/Греции:** натурализация требует **ОТКАЗА от прежнего гражданства** (ст. 8) — Украина отказ разрешает, значит украинский, скорее всего, придётся отдать. + **8 лет** + хорватский язык.
- 🔴 **DN-виза — налоговый тупик:** иностранный доход 0%, но 18 мес без продления, к ПМЖ/гражданству не ведёт.

## 1. Налоги — ✅ уточнено deep-research (2026-06-20)

**Paušalni obrt (фикс-ИП) — ОЧЕНЬ низкий, но с жёстким потолком:**
- Потолок оборота **€60,000/год**.
- База = 15% оборота, налог 12% → подоходный лишь **~1.86% от валовой**; + фикс. взносы **€290.98/мес** (~€3,492/год).
- 🟢 **Эффективно при €58k ≈ ~8%** (ниже, чем я писал ~9–13%; взносы фиксированы → **ставка падает с ростом дохода**, «11–13%» — только на €30–35k). Это **ниже Кипра/Греции** на базовом доходе.
- 🔴 $5,600 (~€58k) впритык под капом; **$7,500 (~€78k) — вылет** → авто-регистрация **НДС 25%** + обычный режим.

**Обычный режим (выше €60k) — средний:**
- Подоходный **15–23%** (до €60k) / **25–33%** выше, по муниципалитету (Загреб max 23/33%; дефолт 20/30%). ⚠️ **Prirez (надбавку) отменили в 2024** — теперь зашита в ставки (прежняя «+18%» неактуальна).
- + взносы пенсия ~20% + медицина ~16.5%.
- Суммарно **~30–40%** для твоего дохода.

**DN-виза:** 🟢 иностранный доход **0% хорватского налога** (даже 183+ дней — ты «налоговый нерезидент»), но 🔴 **18 мес, без продления, тупик** для резиденции/гражданства. ⚠️ Отменяет только хорватский налог; где ты налоговый резидент — отдельный вопрос (DN-«серая зона»).

**Налог: DN vs obrt — суть:** DN = **0%, но тупик 18 мес**; obrt = **~8% (≤€60k), но строит резиденцию** к ПМЖ/гражданству. Для чистого накопления 1.5 года выгоднее DN; для долгого плана — obrt.

## 2. Легализация / путь к гражданству

- **TP** для украинцев (как везде в ЕС, до 04.03.2027).
- **ВНЖ по самозанятости (obrt) ЗАСЧИТЫВАЕТСЯ** в срок резидентства (✅ MUP); временное пребывание/убежище тоже. **DN-виза НЕ считается** (тупик). → цепочка obrt → 5 лет ПМЖ → 8 лет → гражданство технически работает.
- Переход с польской TP на хорватский obrt-ВНЖ в принципе возможен, но **точные пороги дохода/средств для permit'а не установлены** (открытый вопрос).
- **Гражданство: 8 лет** непрерывного резидентства **+ статус ПМЖ** + хорватский язык/латиница + знание культуры (точный CEFR-уровень не зафиксирован).
- 🔴🔴 **ОТКАЗ от прежнего гражданства ОБЯЗАТЕЛЕН — подтверждено (ст. 8).** Исключение «если страна не даёт отказаться» **НЕ помогает**, т.к. Украина отказ разрешает → тебе пришлось бы **сдать украинский паспорт**. (Отказ не нужен по ст.10 брак с хорватом / ст.16 этнический хорват — тебе не подходят.)
- ⚠️ Украина внесла Хорватию в свой список двойного гражданства (Постановление №589, 08.05.2026), **но это нерелевантно** — хорватский закон всё равно заставит отказаться. → **для цели «паспорт ЕС, сохранив украинский» Хорватия ДИСКВАЛИФИЦИРОВАНА.**

## 2-bis. Партнёрша — ✅ признаётся БЕЗ брака в обоих случаях (проверено 2026-06-20)

Хорватия признаёт незарегистрированного партнёра — но с условием **3 лет задокументированных отношений** (или общий ребёнок, или брак). Работает в обоих сценариях:

- **При DN-визе:** можно включить супругу **И незарегистрированного партнёра** (неформальное партнёрство признаётся, case-by-case). Незарегистрированному — доказать ≥3 года отношений. Доход-порог +10% за каждого члена. Партнёрша НЕ получает налоговую льготу DN (но дохода у неё нет → неважно).
- **Без DN (воссоединение на obrt-ВНЖ):** признаётся **izvanbračna zajednica** — заявления обоих + свидетели + совместное проживание **≥3 года** (или ребёнок). Спонсор должен прожить **≥1 года** легально; ВНЖ партнёрше до 2 лет, привязан к твоему.

**Для вас:** незарегистрированный путь возможен, но нужны **3 года отношений на бумаге**; если нет — **брак решает сразу** (вы готовы). То есть брак НЕ обязателен (3 года сожительства — альтернатива), но это самый быстрый вариант.

## 3. Стоимость / уровень жизни

- 🟢 **Климат-парадокс:** тёплое средиземноморское **побережье** (Сплит/Задар/Риека) — но там дороже + летняя наценка; дешёвый **Загреб — континентальный, холодная зима**. Дёшево = холодно, тепло = дороже.
- **Полный бюджет пары (проверено 2026-06-22, Numbeo/Wise):** Загреб (дешевле) — аренда €600–1,000 + коммуналка €120–180 + еда €300–400 → комфортно **~€2,000–2,600/мес**; Сплит/побережье ~€2,400–3,200 (+ летняя наценка). При paušal ~8% (net ~€4,450 на $5,600) откладываешь **~€1,900–2,450/мес** в Загребе.

### Община и врачи (проверено 2026-06-22)
- 🟡🟢 **Украинская община — приличная: ~27,835 (сент. 2025) / «более 30,000»** TP-получателей. **Сопоставимо с Грузией** (~27k), сильно больше Албании (~2,500). Есть госпортал «Hrvatska za Ukrajinu», интеграция. → живое укр. комьюнити, «своих» найти можно.
- 🔴 **Русскоязычная среда слабая:** этнических русских ~1,481 (перепись 2021); русский знают ~4% хорватов. «Свои» = украинские беженцы (между собой укр./рус.), а не местная русскоязычная инфраструктура.
- 🔴 **Врачи по-русски/украински — нет (как в Албании):** частные клиники Загреба/Сплита широко говорят **по-английски**, русско-/украиноязычные врачи не подтверждены. Слабее Грузии (где врачи говорят по-русски). Для неработающей партнёрши: диаспора — да, врач на родном — скорее нет.

## 3-bis. Полный профиль (deep-research 2026-06-22)

**🛂 Легализация (obrt):** порог дохода **1.5× средней ЧИСТОЙ зарплаты ≈ €2,266/мес** (ты проходишь в разы). **ПМЖ через 5 лет** непрерывно (obrt-годы считаются); отлучки ≤10 мес суммарно / ≤6 мес разом. 🔴 Открыто: (1) переход на obrt изнутри vs через консульство — не установлено (уточнять в MUP); (2) польская TP, вероятно, **НЕ засчитывается** в 5 лет (счётчик с первой хорватской карты).

**🌐 IT-инфраструктура (🟢 отлично):** интернет ~133 Мбит/с (Ookla), оптика ~39% и растёт; коворкинги во всех 5 городах. Для удалённой IT-работы — хорошо.

**🛡️ Безопасность + английский (🟢🟢):** очень безопасно (Crime Index 25 / Safety 75); **английский #2 в мире** (EF EPI 617) → жить можно полностью на английском (компенсирует слабую русскоязычность).

**🩺 Медицина — цены умеренные, язык слабый:** частный специалист €70–150; частная страховка €50–150/мес; публичная HZZO ~€90–130/мес. 🔴🔴 Русско-/украиноязычной медицины **нет системно** (омбудсмен отмечает языковой барьер, решение — переводчики); даже англоговорящего врача искать через рекомендации/частные клиники.

**🏙️ Город:** **Сплит** — тёплое море + климат (но дороже, община меньше); **Загреб** — инфраструктура + безопасность + дешевле + больше украинцев (но холодная зима). Чёткого победителя данные не дали.

**⚠️ Осталось НЕ закрыто:** по-городовое распределение укр. общины; сравнение зим/солнца по городам; **покупка недвижимости украинцем-нерезидентом** (правила взаимности, цены, аренда под сдачу).

## 4. Вердикт (уточнён после deep-research)

🟢 **Лучшая дешёвая тёплая EU-база для НАКОПЛЕНИЯ на базовом доходе:** paušal **~8%** при €58k (ниже Кипра/Греции!), ЕС + Шенген + тёплое Адриатическое море + лёгкий славянский язык. Если бы цель была «копить дёшево в ЕС» — Хорватия была бы в топе.

🔴🔴 **Но для ПАСПОРТА — ДИСКВАЛИФИЦИРОВАНА:** натурализация требует **отказа от украинского** (подтверждено), + 8 лет + ПМЖ; paušal ограничен €60k (твой апсайд $7,500 не влезает → ~30–40%); DN-режим — тупик. Проигрывает Кипру/Греции/Словакии (все разрешают двойное украинцу).

**Итог:** отличная для фазы накопления (тепло + дёшево + ЕС), но **не путь к паспорту, если хочешь сохранить украинский**. Развилка: Хорватия = «дёшево копить в тёплом ЕС, но без паспорта»; Кипр/Греция = «дороже налог, но паспорт + сохранить украинский».
