# Словакия — персональный разбор (ЕС; накопление + путь к паспорту)

> Дата: **2026-06-20**. Курс €1 = $1.157. Доход $5,600/мес ≈ €4,840 ≈ €58k/год; $7,500 ≈ €6,480 ≈ €78k/год. Валюта — евро.
> Статус: разбор под профиль (см. [`README.md`](./README.md)). Правила — **проверить у словацкого бухгалтера/юриста**.

## 0. Краткий вывод (TL;DR)

- **ЕС + еврозона + Шенген**, близко к Украине, **словацкий язык славянский → учить легко** (в разы проще греческого).
- 🟢 **Двойное гражданство РАЗРЕШЕНО** (поправка 2022) — натурализуясь, можно сохранить украинский. Лучше Хорватии (там отказ).
- 🔴 **Гражданство — 8 лет** постоянного резидентства (5 если брак со словаком). Дольше Кипра/Греции (7).
- 🔴 **Налог самозанятого ~20–30%** (прогрессив 19/25/30/35% + соцвзносы; пакет консолидации 2026 поднял налоги). Дороже Кипра (~8–12%)/Греции (~14%).
- 🔴 Континентальный климат, холодная зима, моря нет; Братислава дороговата.
- 🟡 Большое присутствие украинцев (TP); словацкий частично взаимопонятен с украинским.

## 1. Налоги — ✅ уточнено deep-research (2026-06-20)

**Поправка к прежней оценке «~20–30%»: налог НИЖЕ, особенно через фирму.**

- **Прогрессив 19/25/30/35% (2026)** применяется к наёмным/общей базе **выше €100k** — тебя НЕ касается.
- **🟢 Живность (živnosť) до €100k оборота → плоский подоходный 15%** (порог поднят с €60k до €100k с 2025). Ты глубоко внутри.
  - Нормативные расходы 60%, **кап €20,000** → база при €58k ≈ €38k → подоходный ~€5,700.
  - Соцвзносы: мин. ~**€303/мес** (соц) + здоровье **16% без потолка** (~€122/мес мин., но растёт с доходом). Здоровье без капа — главный «утяжелитель».
  - **Эффективно ~22–28%** (оценка; end-to-end расчёт не верифицирован — считать с бухгалтером).
- **🟢 Альтернатива — s.r.o. (фирма) + дивиденды ≈ 16.3% (verified):** CIT 10% (до €100k прибыли) + дивиденды 7% (с прибыли 2025+). Чище живности по ставке, но больше админки. **Это лучший низконалоговый маршрут в Словакии.**
- **IP-box** (50% освобождение на софт) — только корпоративный инструмент, фрилансеру-услуговику не подходит.
- ⚠️ 16% health — временно до 31.12.2027; критерий «самозанятости» меняется с 01.01.2026.
- НДС/reverse-charge по иностранным B2B — **не верифицировано**, подтвердить.

## 2. Легализация (украинцу)

- **TP до 04.03.2027** — даёт право работать (в т.ч. самозанятым) + доступ к публичной медицине/образованию.
- **Самозанятость:** trade license (živnosť) + запись в торговый реестр; переход с TP на стандартный ВНЖ — нужны средства ~**€269/мес** на год.
- **ВНЖ для самозанятого** (EU Immigration Portal) — стандартный маршрут.

## 3. Путь к гражданству — 🔴 уточнено: дольше, чем казалось

- **8 лет непрерывного ПОСТОЯННОГО резидентства (trvalý pobyt)** (verified, Act 40/1993 §7). 🔴 **Ни TP, ни временный ВНЖ самозанятого НЕ считаются** в эти 8 лет. Сам ПМЖ достигается обычно через **~5 лет** → **реальный срок от приезда до паспорта ~10+ лет** (не 8). Хуже Кипра/Греции (7).
- Сокращённые сроки (5 — брак со словаком, 4 — беженец, 3 — словак-зарубежный) тебе не подходят.
- **Язык:** экзамен перед комиссией — интервью по истории/географии + чтение статьи 500+ слов + письменный пересказ. 🟢 Словацкий (западнославянский) **легко** для украинца — реальный плюс против греческого B1.
- ⚠️ **Двойное гражданство (сохранить украинский) — НЕ верифицировано** в этом прогоне (поправка 2022 и укр. список 4502-IX не подтверждены). Перепроверить — это критично для цели.

## 4. Стоимость жизни (Братислава, на двоих)

Wise/Numbeo 2026: один ~€1,550/мес; аренда 1BR центр ~€830, вне центра ~€620; коммуналка ~€183. Пара реалистично **~€2,000–2,400/мес** — дороже Балкан и Грузии, но дешевле Запада ЕС.

## 5. Уровень жизни

- 🔴 **Климат:** континентальный, холодная зима, моря нет.
- 🟡 **Сообщество:** большое присутствие украинцев (TP-беженцы); словацкий близок к украинскому → бытовая адаптация легче; русско-/украиноязычная медицина — не подтверждено (вероятно через украинскую общину).
- 🟢 ЕС/Шенген, безопасно, рядом с Украиной (граница).

## 6. Вердикт (уточнён после deep-research)

🟡 **Налог лучше, гражданство хуже, чем казалось.**
- 🟢 Плюсы: **налог ~16% через s.r.o.** (или ~22–28% живность) — терпимо; **лёгкий славянский язык**; ЕС + Шенген + близко к Украине.
- 🔴 Минусы: гражданство **~10+ лет реально** (8 лет ПМЖ, TP не считается) — **дольше Кипра/Греции (7)**; холодно, моря нет; двойное гражданство пока не подтверждено.

**Под цель «паспорт ЕС, сохранив украинский»:** Словакия проигрывает Кипру/Греции по главному — **сроку (~10+ vs 7 лет)**. Её козырь (лёгкий язык) не перекрывает +3 года. Как накопительная EU-база с понятным языком — ок, но для самого паспорта — медленнее.
