# Сербия — персональный разбор

> Дата: **2026-06-20**. Курсы: **$1 ≈ RSD 101**, **€1 ≈ RSD 117** (динар управляемо привязан к евро), €1 = $1.157. Доход $5,600/мес ≈ RSD 6.8m/год; $7,500 ≈ RSD 9.1m/год.
> Статус: разбор под профиль (см. [`README.md`](./README.md)). Правила — **проверить у сербского бухгалтера/юриста**.

## 0. Краткий вывод (TL;DR)

- 🔴 **Главная засада: льготный paušal тебе НЕ подходит по доходу.** Paušal (фикс-ставка ~€200–500/мес для IT) доступен только при обороте **до RSD 6m/год (~$55k)**. Твои $5,600/мес = ~$67k — **выше потолка**. $7,500/мес = $90k — далеко выше.
- **Реальный режим для тебя — preduzetnik с бухгалтерией:** ~10% подоходного на прибыль + соцвзносы → эффективно ориентировочно **~15–20%**. Это **хуже** Грузии/Албании/Армении, ближе к Болгарии.
- ✅ **Сильный плюс по партнёрше: Сербия признаёт ГРАЖДАНСКИЙ брак** (vanbračna zajednica) для воссоединения семьи — **официальный брак НЕ обязателен** (как в Греции, в отличие от Грузии/Албании/Молдовы/Румынии).
- **Доступно украинцу:** безвиз, TP-рамка для украинцев, **single permit** (ВНЖ+работа) по самозанятости/компании до 3 лет, **ПМЖ через 3 года**.
- Не ЕС (кандидат), но дёшево (особенно Нови-Сад), близко, без моря, динар стабильнее лея/драма.

## 1. Налоги

**Резидентство:** 183+ дня → сербский налоговый резидент.

🔴 **Порог paušal — ключевой вопрос:**
- **Paušal (фикс-ставка):** покрывает подоходный + соцвзносы; для IT (программирование/консалтинг) ~**€200–500/мес**. Доступен при обороте **≤ RSD 6m/год (~€51k / ~$55k)**. Продлён до 2028.
- 🔴 Твои **$67k/год > RSD 6m** → **paušal недоступен** на твоём доходе. Это убирает главную «фишку» Сербии.
- ⚠️ **«Тест независимости» (independence test):** даже если бы доход проходил, работа фактически на одного иностранного клиента может привести к переквалификации преприёмника → доп-нагрузка. Для одноклиентского B2B это риск.

**Что остаётся для твоего дохода:**
- **Preduzetnik с двойной бухгалтерией:** ~10% подоходного с прибыли + обязательные соцвзносы (пенсия/здоровье) → эффективно ориентировочно **~15–20%** (зависит от вычета расходов, базы взносов). Точно — к бухгалтеру.
- **Freelancer-модели (A/B):** для физлиц с иностранным доходом, но там свои квартальные пороги и они не дают paušal-выгоды на твоём уровне.
- **НДС:** регистрация при обороте > RSD 8m/год (~$79k) — при росте до $7,500 ты приближаешься; экспорт услуг обычно 0%/вне сферы — подтвердить.

**Оценка ($5,600/мес, preduzetnik ~15–20%):** на руки ~**$4,500–4,800/мес**.

## 2. Легализация (под украинца)

- **Безвиз** для украинцев + **рамка временной защиты** для украинцев в Сербии.
- **Single permit (обединена дозвола)** — объединяет ВНЖ и право на работу; для **самозанятости / основателя компании**; выдаётся до **3 лет**.
- **ПМЖ через 3 года** непрерывного законного пребывания (допускаются отлучки до 10 мес суммарно / 6 мес разово), далее возможно гражданство.
- 🔴 **Пост-2027:** Сербия-специфичный мост со старой TP в обычный ВНЖ в досье не зафиксирован, но **single permit по самозанятости не зависит от TP** — это и есть устойчивый путь.
- 🔴 **Не ЕС** (кандидат) → евро-ПМЖ напрямую не строит, но сербский ПМЖ реален.

## 3. Девушка / будущая жена — сильная сторона Сербии

✅ **Воссоединение семьи покрывает и гражданский брак** (vanbračna zajednica), не только официальный:
- Для брака — свидетельство о браке.
- Для гражданского союза — нотариальные заявления обоих партнёров + заявления **двух свидетелей** + доказательства совместной жизни (общий адрес и т.п.).
- ВНЖ по воссоединению — до 3 лет, продлевается; через 3 года → ПМЖ.

То есть партнёрше **не обязательно становиться женой** — это редкий плюс среди твоих стран (так умеет только Греция).

## 4. Стоимость жизни (на двоих)

Livingcost/Numbeo (2026): Белград ~$1,378/мес база (один без аренды ~€615); **Нови-Сад ~на 19% дешевле Белграда**; Ниш ещё дешевле.

| Город | Аренда 1–2BR | Реалистичный бюджет (пара) |
|---|---:|---:|
| **Нови-Сад** | $450–650 | ~$1,700 (лучший баланс) |
| Белград | $550–900 | ~$2,100 (давление аренды) |
| Ниш | $350–500 | ~$1,550 (дешевле всех) |

При preduzetnik (net ~$4,500–4,800) и Нови-Саде накопление ~**$2,800–3,000/мес** → **$90k за ~30–32 мес**.

### Климат — 🔴 континентальный, не средиземноморский (минус против Албании)

Холодная зима, жаркое лето, 4 сезона, **моря нет**:
- **Зима:** Белград днём ~6°C; Нови-Сад ~4°C, в холода до **−15°C**, снег/иней → выше счета за отопление.
- **Лето:** Белград до 31°C; Нови-Сад до 35°C, душно.
- Юг (Ниш) чуть мягче летом, но зима всё равно холодная.

Для приоритета «тёплый климат» — слабо (противоположность тёплому побережью Албании).

### Покупка квартиры в Сербии (цены 2026)

Цена/м²: **Нови-Сад ~€1,800–2,250**, **Белград ~€2,300–2,560**. 3–4 комн. (~85 м²): Нови-Сад **~€150–190k**, Белград **~€200–270k**.

| Цель | Сумма | Срок при ~$2,950/мес |
|---|---:|---:|
| Капитал/первый взнос $90k | $90k | ~31 мес |
| 3–4 комн. Нови-Сад за кэш | ~€170k | ~67 мес |

⚠️ Покупка в Сербии = недвижимость **вне ЕС** → цели «осесть в ЕС» не служит. Если конечная квартира в ЕС — в Сербии **копить**, покупать уже резидентом ЕС (см. [`buying-property-in-eu.md`](./buying-property-in-eu.md)).

## 5. Уровень жизни

Плюсы: **сербский близок, к русскому/украинцам дружелюбно**, дёшево (вне Белграда), кафе-культура, безопасно, динар стабильнее лея/драма, ЕС-кандидат.
Минусы: **не ЕС** (вступление не близко); налог для твоего дохода средний (paušal не достаётся); Белград — давление аренды; нет моря; зимой смог в некоторых городах; политический фон.

## 7. Вердикт (первичный)

Сербия — **середняк по накоплению, но сильна по гибкости статуса**:
- 🔴 По чистой скорости накопления **слабее**, чем казалось: paušal недоступен на твоём доходе → эффективно ~15–20%, уровня Болгарии, а не Грузии.
- ✅ Зато **3-летний ПМЖ + признание гражданского брака** (партнёрше не нужен официальный брак) + лёгкий доступ украинцу + дёшево в Нови-Саде.

**Кому подходит:** если приоритет — **не максимальная скорость кэша, а гибкий путь к ПМЖ без обязательного брака и комфортная дешёвая база рядом с ЕС**. Для чистого «копить максимально быстро» — Грузия/Албания/(Армения) лучше. Для ЕС-ПМЖ — Греция.

## 8. Открытые вопросы (бухгалтеру/юристу)
1. **Главный:** реальная эффективная ставка preduzetnik с бухгалтерией на твоём доходе (10% + взносы, вычет расходов, база); точно ли paušal закрыт > RSD 6m.
2. «Тест независимости» для одноклиентского иностранного B2B — риск переквалификации и как структурировать.
3. НДС/экспорт услуг, порог RSD 8m при росте до $7,500.
4. Single permit по самозанятости: точный чек-лист для украинца, считается ли в 3 года ПМЖ.
5. Гражданский брак (vanbračna zajednica): практика признания для воссоединения, какие доказательства реально принимают.
